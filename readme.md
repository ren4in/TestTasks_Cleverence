# Тестовое задание

Репозиторий содержит 3 задачи на C# / .NET:

- Задача 1 — сжатие и распаковка строки  
- Задача 2 — потокобезопасный счётчик  
- Задача 3 — стандартизация лог-файлов

## Задача 1 — Сжатие строки

Реализованы:

- Compress — сжатие строки (aaabb → a3b2)
- Decompress — восстановление строки

Особенности:
- поддержка многозначных чисел (a12)
- валидация входных данных

Запуск:
сделать Task1.Compression стартовым проектом и запустить

---

## Задача 2 — Потокобезопасный счётчик

Статический класс ServerCounter:

- GetCount — чтение
- AddToCount — запись

Используется ReaderWriterLockSlim:
- параллельное чтение
- эксклюзивная запись

Запуск:
сделать Task2.ConcurrentServer стартовым проектом

---

## Задача 3 — Стандартизация логов

Консольная утилита, которая:

- читает лог-файл
- приводит записи к единому формату
- сохраняет:
  - валидные строки → output.txt
  - невалидные → problems.txt

Поддерживаются 2 формата логов.

Архитектура:

- LogParser — парсинг строки
- LogLevelMapper — нормализация уровня
- LogEntryFormatter — форматирование
- LogFileProcessor — обработка файла

Запуск:
сделать Task3.LogStandardizer стартовым проектом
